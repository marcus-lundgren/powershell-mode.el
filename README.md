# powershell-mode.el
A simple powershell highlighting mode.

To load from a local copy:

```lisp
(use-package powershell-mode
  :load-path "~/path/to/powershell-mode.el/")
```

To load from GitHub:

```lisp
(use-package powershell-mode
  :vc (:url "https://github.com/marcus-lundgren/powershell-mode.el"
            :rev :newest))
```
